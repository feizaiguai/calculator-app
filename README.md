# Calculator App 计算器应用

Build a Calculator Application with the following features:
1. Standard calculator with +, -, ×, ÷ operations
2. Scientific calculator mode with sin, cos, tan, log, ln, sqrt, power, factorial
3. Expression input with full keyboard support
4. Calculation history panel showing all previous calculations
5. Click on history item to reuse the result
6. Memory functions: MC, MR, M+, M-
7. Percentage and sign toggle (+/-)
8. Parentheses support for complex expressions
9. Decimal precision control
